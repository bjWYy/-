第一步：确认题库

系统已内置你上传的题库，可直接开始练习或考试；也支持继续导入新的 JSON 题库覆盖当前数据。

查看题库格式说明

系统当前支持三类题目：

  • single：单选题
  • multiple：多选题
  • truefalse：判断题

每道题建议包含以下字段：

{
  "type": "single",
  "question": "题目内容",
  "options": ["选项A", "选项B", "选项C", "选项D"],
  "answer": ["B"],
  "knowledgePoint": "知识点",
  "explanation": "解析内容",
  "source": "品牌或分类"
}

其中 answer 统一写成数组；单选题填写一个字母，多选题可填写多个字母，例如 ["A", "C"]。

第二步：选择模式

练习模式

随机抽题。作答后立即显示正确答案、知识点与解析。

考试模式

随机抽取 100 题，全部完成后评分。90 分为及格线，并提供逐题解析。
