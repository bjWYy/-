# 题库练习与考试系统（后台管理版）

## 功能
- 练习模式：随机抽题，作答后立即显示答案、知识点、解析
- 考试模式：随机抽取 100 题，全部完成后评分，90 分及格
- 后台管理：管理员登录后可新增、编辑、删除、搜索、筛选、导入、导出题库
- 数据持久化：浏览器本地保存后台修改结果

## 使用方法
1. 直接打开 `index.html`
2. 默认管理员密码：`admin123`
3. 如需修改管理员密码，请编辑 `script.js` 中的 `DEFAULT_ADMIN_PASSWORD`
4. 可部署到 GitHub Pages / Netlify / Vercel 生成公网网址

## 说明
- 当前版本为前端静态后台，适合中小规模内部题库使用
- 后台数据默认保存在浏览器本地 `localStorage`
- 若你要多账号、云端同步、考试记录留存、学员登录、排行榜，需要再接后端数据库
